Chapter 17: HTML Layout

  1. The new HTML5 elements indicate the purpose of different parts of a web page and help to describe its structure:
    • < header > Defines a header for a document or a section.
    • < nav > Defines a set of navigation links.
    • < section > Defines a section in a document.
    • < article > Defines an independent, self-contained content.
    • < aside > Defines content aside from the content .
    • < footer > Defines a footer for a document or a section.
  2. The new elements provide clearer code (compared with using multiple
